# Police Station Hierarchy - Sovereign Ledger UI

A high-trust, minimal React dashboard for visualizing and managing police unit hierarchies.

## ✨ Features
* **Hierarchy Visualization:** Clear, nested tree-view of reporting structures.
* **Dynamic Management:** Seamlessly add or edit units at any level.
* **Real-time Updates:** UI reflects CRUD changes instantly without reload.
* **Analytics Dashboard:** Visual cards for unit health and compliance rates.

## 🛠️ Tech Stack
* **Framework:** React.js
* **Styling:** Tailwind CSS (Bento Grid layout)
* **Icons:** Lucide-React
* **State/API:** Axios for secure backend communication.

## 📥 Setup
1. Clone the repo.
2. Run `npm install`.
3. Update `BASE_URL` in your API config to point to the hosted backend.
4. Run `npm run dev` to start the local development server.
